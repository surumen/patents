Patents API

This API contains patent applications data fetched from the USPTO API. It is designed to simplify your work so that you only focus on developing your front-end client.

BASE_URL
http://patentsapi-dev.us-east-2.elasticbeanstalk.com/api/patents/

Sample Data

The USPTO api provides the following information about the patent applications on their database.

Parameter Data Type
applicationType String
documentId String
applicationNumber String
documentType String
publicationDate String
documentDate String
productionDate String
applicationDate String
applicant List of String(s)
inventor List of String(s)
assignee List of String(s)
title String
archiveUrl String
pdfPath String
year String
_version_ Long

Get all Patents

You can view upto 100 patent applications at a time. Our API refreshes this data every 24 hours.

GET
/api/patents/

Example request

client.listPatents(function(err, patents) {
  console.log(patents);
});

Example response

{
 "_embedded" : {
    "patents" : [ {
        "applicationType": "{applicationType}",
        "documentId": "{document_id}",
        "applicationNumber": "{application_number}",
        "assignee": "{assignee}"
        ...
      },
      ...
    ]
}

Retrieve a Single Patent

Retrieve information about just one existing patent. The patents are identified by a number from 1 to 100.

GET
/api/patents/{number}

Example request

client.readPatent('patent-number',
  function(err, patent) {
    console.log(patent);
  });

Example response

{
  "applicationType": "{type}",
  "documentId": "{doc_id}",
  "applicationNumber": "{app_num}",
  "documentType": "{doc_type}",
  "publicationDate": "{date}",
  "documentDate": "{date}",
  "productionDate": "{date}",
  "applicationDate": "{date}",
  "applicant": "{[applicant1, applicant2, ...]}",
  "inventor": "{[inventor1, inventor2, ...]}",
  "assignee": "{[assignee1, assignee2, ...]}",
  "title": "{title}",
  ...
}
